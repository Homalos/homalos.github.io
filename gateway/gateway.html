网关和外部连接 ​

字数
1160 字
阅读时间
6 分钟

本文档涵盖了 Homalos 连接外部交易平台和市场数据源的网关层。网关系统提供了跨多个交易平台的市场数据订阅、订单执行和账户管理的抽象接口。

有关网关如何与核心交易引擎交互的信息，请参阅[交易引擎]。有关网关使用的事件驱动通信模式的详细信息，请参阅[事件驱动系统]。

网关架构概述 ​

Homalos 网关系统采用双网关架构，每个外部平台需要两个专用网关：一个用于市场数据，一个用于订单执行。这种分离机制实现了独立的扩展和故障隔离。

市场数据网关实施 ​

市场数据网关负责处理实时市场数据订阅和报价数据处理。CTP 和 TTS 的实现都遵循类似的模式，并使用特定于平台的 API 包装器。

市场数据订阅管理 ​

CTP 市场数据网关通过连接状态验证和待处理请求排队实现了复杂的订阅管理：

元素目的关键方法
pending_subscriptions跟踪等待订阅的符号set[str]
active_subscriptions跟踪成功订阅的符号set[str]
pending_subscription_queue断开连接期间排队请求list[dict]
ConnectionState管理连接生命周期DISCONNECTED，CONNECTING，CONNECTED，LOGGED_IN

订阅过程包括验证和重试逻辑：

def _handle_gateway_subscribe(self, event: Event) -> None:
    if not self._is_gateway_ready():
        self._queue_pending_subscription(strategy_id, symbols)
        self._trigger_reconnection()
        return
    
    for symbol in symbols:
        if symbol not in self.active_subscriptions:
            subscribe_req = SubscribeRequest(symbol=symbol, exchange=self._get_symbol_exchange(symbol))
            self.pending_subscriptions.add(symbol)
            self.subscribe(subscribe_req)

async def _auto_reconnect_loop(self) -> None:
    while (self._current_reconnect_attempts < self._max_reconnect_attempts and 
           self.connection_state == ConnectionState.DISCONNECTED):
        
        self._current_reconnect_attempts += 1
        wait_time = min(self._reconnect_interval * (2 ** (self._current_reconnect_attempts - 1)), 60)
        
        await asyncio.sleep(wait_time)
        
        if self._last_connection_config:
            self.connect(self._last_connection_config)

async def _heartbeat_monitor_loop(self) -> None:
    while self._running and self.connection_state != ConnectionState.DISCONNECTED:
        await asyncio.sleep(30)
        
        current_time = time.time()
        if self.last_heartbeat > 0 and current_time - self.last_heartbeat > 60:
            logger.warning("心跳超时，检测到连接断开")
            await self._handle_connection_lost()

def _set_gateway_state(self, new_state: GatewayState) -> None:
    with self._state_lock:
        old_state = self._gateway_state
        self._gateway_state = new_state
        
        self._safe_publish_event("gateway.state_changed", {
            "gateway_name": self.gateway_name,
            "old_state": old_state.value,
            "new_state": new_state.value
        })
