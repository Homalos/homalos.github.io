启动方式 ​

Homalos 支持一键启动，执行启动命令之前请确保激活当前 Python 虚拟环境：

bash
.venv\Scripts\activate

启动主程序：

bash
python -m start_homalos

一键启动干了哪些事情：

  • 初始化配置管理器
  • 初始化事件总线
  • 初始化服务注册中心
  • 初始化数据服务(含行情和数据持久化)
  • 初始化交易引擎核心(含策略管理、风控模块、订单管理)
  • 初始化Web管理界面
